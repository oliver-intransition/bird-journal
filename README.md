This project was inspired by problems I faced whilst trying to record my birdwatching from a previous trip. Specifically, I was frustrated by the need to start a session when out birdwatching using eBird to record the location of the birds I was seeing. I wanted to create something that would sync between my devices, but more importantly allow me to retrorespectively record the birds I was seeing in specific locations, whilst also compiling a total 'life list'.
